## Build Setup

``` bash
# install dependencies
# 安装依赖
yarn

# serve with hot reload at localhost:8080
# 启动开发服务器
yarn dev

# run unit and e2e tests
# 运行单元测试和端对端测试
yarn test

# build for production with minification
# 打包并压缩
yarn build
```
